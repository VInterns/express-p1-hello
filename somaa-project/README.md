# Express-app1-hello
A Simple Express App that implements a GET method to return a JSON Object.

## Prerequisites
* NodeJs
* ExpressJS
* Nodemon

### How To Use

```
$ git clone https://github.com/VInterns/express-p1-hello
$ cd express-p1-hell
$ cd somaa-project
$ npm install
$ npm start
```
Then open a browser and go to localhost:3000/hello.
